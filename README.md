# 🐾 Desktop Pet

A fun and interactive **Desktop Pet** built using **HTML, CSS, and JavaScript**. The pet lives on your screen, reacts to user interactions, performs animations, and brings life to your desktop through an engaging browser experience.

---

## ✨ Features

- 🐱 Interactive desktop pet
- 🎭 Multiple animations and expressions
- 🖱️ Responds to user interactions
- 🎮 Built-in mini interactions
- 🌙 Smooth animations and transitions
- 📱 Responsive design
- ⚡ Lightweight and fast
- 🚀 Runs directly in your browser

---

## 🛠️ Technologies Used

- **HTML5** – Structure
- **CSS3** – Styling & Animations
- **JavaScript (ES6)** – Functionality & Interactivity

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/usaidkhot159/Desktop-pet.git
```

### Open the project

Open `index.html` in your favorite web browser.

No installation or additional setup is required.

---

## 🎮 How to Use

- Launch the project by opening `index.html`.
- Interact with the pet using your mouse.
- Watch the pet react with different animations and behaviors.
- Explore the various interactive features available.
